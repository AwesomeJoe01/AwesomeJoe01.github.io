List Tags

Tag Description Example
<ul> Creates an unordered list.
Each list item will start with a bullet point.
  • First
  • Second
  • Third
<ol> Creates an ordered list.
Each list item will start with a sequential numeric value.
  1. First
  2. Second
  3. Third
<li> Adds elements to a list, ordered or unordered.
  1. Ordered element
  • Unordered element
