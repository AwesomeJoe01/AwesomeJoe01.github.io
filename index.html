What is HTML, CSS, and JS?

HTML5 (Hypertext Markup Language Version 5) creates the structure of each page on the web. CSS3 (Cascading Style Sheets Level 3) allows developers to add simple yet powerful styling to an HTML document. Modern Javascript is a programming language that facilitates interactive, responsive websites. Together, these make up the DOM, or the Document Object Model, which creates a hierarchy tree after parsing a document.
